# SmartBox Hardware

This folder contains the hardware artifacts for the DriveWise SmartBox (v0).

## Contents

### Architecture
Photos of the SmartBox hardware design and physical layout v0.

### Assembly
CAD assembly files and component models:
- Native SolidWorks files (`.SLDASM`, `.SLDPRT`)
- STEP/STP files for cross-platform viewing
- Component models (Raspberry Pi 4, Arducam IMX519, case covers, etc.)

## Tools

- **SolidWorks** 
- **FreeCAD**
